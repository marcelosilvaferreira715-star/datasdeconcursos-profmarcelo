# datasdeconcursos-profmarcelo

Projeto React + Vite + Tailwind com a lista de datas de concursos.

## Instruções rápidas

### Rodar localmente
1. Tenha Node.js (v18+) instalado.
2. No terminal:
```bash
npm install
npm run dev
```
3. Abra `http://localhost:5173`

### Deploy no Vercel (upload do ZIP)
1. Crie conta em https://vercel.com (recomendado com GitHub ou e-mail).
2. No painel do Vercel clique em **New Project** -> **Import** ou use **Upload**.
3. Se fizer upload do ZIP, escolha o arquivo `datasdeconcursos-profmarcelo.zip`.
4. O Vercel detectará o projeto Vite e fará build automaticamente.
5. Após deploy você terá o link do site (algo como https://datasdeconcursos-profmarcelo.vercel.app).

### Observações
- A senha de edição é salva localmente no navegador (localStorage). Se quiser edição centralizada em vários dispositivos, posso configurar integração com Google Sheets ou Firebase.
- O formulário de contato utiliza Formspree (primeiro envio pedirá verificação do e-mail `marcelomferreira2025@gmail.com`).
